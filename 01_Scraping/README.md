# 01_Scraping - Origen de los Datos

## Descripción
Esta carpeta está reservada para los scripts o procesos de recolección de datos. 

## Procedencia de los Datos
Para este proyecto, los datos no fueron obtenidos mediante *scraping* automatizado en tiempo real, sino que fueron provistos como datasets estructurados (`.csv`).

Si en futuras iteraciones del proyecto se automatiza la recolección de datos desde fuentes externas (como APIs de la FIFA o sitios de estadísticas deportivas), los scripts correspondientes serán alojados en este directorio.
