# Predicciones - Resultados de Simulación

Esta carpeta contiene los productos finales generados por los modelos probabilísticos.

## Archivos generados:
* `predicciones_monte_carlo_mundial.csv`: 
    - **Propósito:** Probabilidad estadística.
    - **Explicación:** Resultado de 1,000 iteraciones del torneo. Indica qué porcentaje de veces cada equipo se coronó campeón, útil para determinar al equipo "favorito" bajo condiciones de azar.
    
* `detalle_simulacion_torneo.csv`: 
    - **Propósito:** Trazabilidad partido a partido.
    - **Explicación:** Muestra el detalle técnico de una simulación completa (el "camino al título"). Incluye los goles esperados (xG) para ambos equipos, el marcador final simulado y cómo se resolvió cada enfrentamiento.
