# 03_Modelado_Simulacion

Aquí se reside el núcleo de la inteligencia artificial del proyecto.

## Contenido:
- **`03_Modelo.ipynb`**: Notebook dedicado al entrenamiento del algoritmo:
    - **Algoritmo:** `RandomForestRegressor`.
    - **Variables de entrada:** Métricas de rendimiento histórico (xG, posesión, etc.).
    - **Output:** Esperanza matemática de goles para equipo local y visitante.
    - **Evaluación:** Incluye métricas de rendimiento como MAE, RMSE y R².
