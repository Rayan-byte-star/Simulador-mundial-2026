# Data - Repositorio de Insumos

Esta carpeta contiene los datasets crudos utilizados para alimentar el motor de Machine Learning.

## Archivos incluidos:
* `datos_mundial.csv`: Estadísticas de rendimiento (xG, posesión, tiros) de los equipos participantes en el mundial.
* `datos_historicos.csv`: Historial de partidos previos usado para entrenar el modelo.
* `partidos.csv`: Base de datos maestra de enfrentamientos.
* `Grupos_Mundial.csv`: Estructura oficial de los grupos del torneo.
* `ranking_fifa.csv`: Posiciones oficiales de la FIFA para ponderar la fuerza de los equipos.
* `transfermarkt.csv`: Valor de mercado de los equipos como variable económica de referencia.
